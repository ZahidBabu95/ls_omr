# LS_OMR (AmarSchool OMR SaaS) - Project Structure Analysis

This document provides an overview of the current project structure for the **AmarSchool OMR SaaS Platform** built using Laravel.

## 📌 Technology Stack
* **Backend:** Laravel 11.x (PHP 8.2+)
* **Frontend:** Blade Templates, Tailwind CSS, Vanilla JS
* **Database:** SQLite / MySQL 

## 📂 Core Directory Structure

### `app/` (Application Logic)
* **`Http/Controllers/`**
  * `LandingController.php`: Manages the frontend landing page logic, mapping DB to views.
  * `ChatbotController.php`: Handles AI chatbot interactions (TBI).
  * **`Admin/`** (Backend Administrative Controllers):
    * `SettingController.php`: System-wide UI configs.
    * `LeadController.php`: Waitlist requests management.
    * `TestimonialController.php`: Manages client reviews and avatars/images.
    * `PricingPlanController.php`: Dynamic BDT pricing builder mapping JSON feature arrays.
    * `FaqController.php`: FAQ list sequencing and display.
    * `DownloadController.php`: Handles secure file uploads for OMR sheet templates.
* **`Models/`**
  * Contains Eloquent ORM models defining the data structure: 
  * `User`, `Feature`, `Setting`, `Faq`, `PricingPlan`, `Download`, `Lead`, `Testimonial`.

### `database/`
* **`migrations/`**
  * Complete SaaS schema migrations inclusive of newly appended images features (`add_image_to_testimonials_table`).
* **`seeders/`**
  * `DatabaseSeeder.php`: Populates initial values.

### `resources/` (Frontend & Admin UI)
* **`views/welcome.blade.php`**: The primary full-featured Landing Page (Highly responsive layout, dynamic integrations).
* **`views/layouts/navigation.blade.php`**: Sidebar structured for the admin application.
* **`views/admin/`**: Contains segmented directories mapping accurately to each CRUD function (e.g., `faqs/index`, `faqs/create`). Includes file multipart uploads for testimonials/downloads.

### Configuration & Tooling
* `tailwind.config.js`: Configuration for UI styling.
* `storage/app/public/`: Root directory storing dynamic reviewer templates and downloadable assets.
